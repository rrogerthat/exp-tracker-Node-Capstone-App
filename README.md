<h1>Wallet Tracker</h1>
<p>CRUD API App</p>

<p>A simple-to-use app to track your daily expenditures where you can add, delete, and edit your expenses and see the total amounts spent by category.</p>

<p><b>Link:</b> https://node-exp-tracker.herokuapp.com/</p>

<h3><u>Technology</u></h3>

<h4>Front-end:</h4>
<li>HTML</li>
<li>CSS</li>
<li>Javascript</li>
<li>JQuery</li>
<li>Front-end</li>

<h4>Back-end:</h4>
<li>Node.js</li>
<li>Express.js</li>
<li>MongoDB</li>
<li>Mongoose</li>
<li>mLab</li>

<h4>Authentication:</h4>
<li>Passport.js</li>
<li>Passport.jwt</li>
<li>Bcryptjs</li>

<h4>Testing:</h4>
<li>Chai</li>
<li>Mocha</li>
